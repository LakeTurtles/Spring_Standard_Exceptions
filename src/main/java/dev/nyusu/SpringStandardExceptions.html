Java Spring Standard Exceptions with HTTP Status Codes

Exception Description HTTP Status Code
DuplicateKeyException Thrown when an attempt to insert or update data results in a duplicate key violation. 409 Conflict
InvalidDataAccessApiUsageException Thrown when incorrect usage of Spring's data access API is detected.
IncorrectResultSizeDataAccessException Thrown when a result was expected to have a certain size but the returned result set has a different size.
CannotCreateTransactionException Thrown when a transaction manager cannot create a new transaction or suspend an existing transaction.
TransactionSystemException Thrown when the underlying transaction system fails unexpectedly.
UnexpectedRollbackException Thrown when the outcome of a transaction is unexpected.
InvalidPropertyException Thrown when trying to set an invalid property value.
IllegalStateException Thrown when an IllegalStateException is encountered in a Spring component.
BeanCreationException Thrown when a bean is not able to be created.
BeanDefinitionStoreException Thrown when a bean definition cannot be loaded or registered.
NoSuchBeanDefinitionException Thrown when a requested bean definition is not found.
ConversionNotSupportedException Thrown when an attempt to convert a value to another type fails.
NoSuchRequestHandlingMethodException Thrown when no suitable request-handling method is found.
HttpMediaTypeNotAcceptableException Thrown when the server is unable to produce a response matching the list of acceptable values defined in the request's headers. 406 Not Acceptable
HttpMediaTypeNotSupportedException Thrown when the server refuses to accept the request because the request entity is in a format not supported by the requested resource. 415 Unsupported Media Type
HttpMessageNotReadableException Thrown when the request body could not be read. 400 Bad Request
HttpMessageNotWritableException Thrown when the request handler is unable to write to the response. 500 Internal Server Error
HttpServerErrorException Thrown when a server error is encountered, such as a 5xx status code.
HttpClientErrorException Thrown when a client error is encountered, such as a 4xx status code.
HttpStatusCodeException Thrown for HTTP 4xx and 5xx status codes.
MissingServletRequestParameterException Thrown when a request handler expects a certain parameter, but it's missing. 400 Bad Request
ServletRequestBindingException Thrown when there's an error related to request binding.
MethodArgumentNotValidException Thrown when method argument validation fails. 400 Bad Request
HttpMessageConversionException Thrown when an HTTP message cannot be converted. 500 Internal Server Error
TypeMismatchException Thrown when an attempt to set a bean property with an invalid value is detected.
MethodNotAllowedException Thrown when a request handler does not support a specific request method. 405 Method Not Allowed
